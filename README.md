# BJTU Course Assistant

北京交通大学课程助手浏览器扩展。

## 功能简介

- 课程列表查看
- 课件、回放下载
- 作业聚合展示（智慧课程平台、雨课堂、每日交作业、接龙管家）
- 作业状态区分（待完成、已提交）
- 智慧课程平台作业提交
- 常用页面快速跳转
- 快速切换账号
- 查看资源空间

## 安装方式

1. 打开浏览器扩展管理页面(about://extensions)。
2. 开启开发者模式。
3. 点击“加载已解压的扩展程序”，选择本目录。

## 使用说明

1. 在扩展页面填写学号并登录智慧课程平台。
2. 按需启用其他平台。
3. 等待课程与作业加载完成后进行更多操作。
