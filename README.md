# BJTU Course Assistant

北京交通大学课程助手浏览器扩展。

## 功能简介

- 课程列表查看
- 作业聚合展示（智慧课程平台、雨课堂、每日交作业、接龙管家）
- 作业状态区分（待完成、已完成）
- 常用页面快速跳转

## 安装方式

1. 打开浏览器扩展管理页面（Chrome 可使用 `chrome://extensions`）。
2. 开启开发者模式。
3. 点击“加载已解压的扩展程序”，选择本目录。

## 使用说明

1. 在扩展页面填写学号并登录。
2. 选择需要启用的平台。
3. 等待课程与作业加载完成后进行查看和跳转。
